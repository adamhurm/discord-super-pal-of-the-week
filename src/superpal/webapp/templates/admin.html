Admin Dashboard — Guide

Excluding a Member

Click Exclude next to a member to remove them from the card draw pool. Excluded members will not appear in /draw-card, /trade-in, or /upgrade results. Click Re-include to restore them.

Custom Members

Members added via Add Member Manually are marked CUSTOM and are never modified by the Discord sync. Their display name, avatar, rarity lock, and exclusion status are preserved exactly as set here regardless of how many times you sync.

Rarity Lock

Only CUSTOM members have a rarity dropdown. Use it to restrict which rarity tier that member can appear in — e.g. set to Legendary only so they only show up on a Legendary roll. Real Discord members always follow normal rarity rules. Set to Any rarity to remove the restriction.

Syncing the Member List

Click Sync Member List from Discord to pull the current server roster into the database. Do this after members join or change their display name so their cards stay up to date.

Card Pool Stats

Eligible — members currently in the draw pool.
Excluded — members blocked from the pool.
Cards in circulation — total cards held across all players (counts duplicates).

Resetting Weekly Draw Counts

Click Reset Weekly Draw Counts to clear everyone's draw usage for the current week. Players who have already used their draws this week will be able to draw again immediately. Historical weeks are not affected.

Accessing This Page

Run /admin-link in Discord (requires The Clippy role). The bot DMs a one-time link — first click opens a 24-hour admin session.
